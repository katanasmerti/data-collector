# Data Collector Angular App with WebWorker

Using Angular 14 and Bootstrap 5.2.

## Development

Install all dependencies.

```
npm i
```

Start serving the application to `http://localhost:4000`.

```
npm run start
```

Build the application to `dist/`.

```
npm run build
```

Check code quality.

```
npm run lint
```
